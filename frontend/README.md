# Smart Hospital Frontend

This is a Vite + React + TailwindCSS starter scaffold for the Smart Hospital Management System frontend.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

3. Build

```bash
npm run build
```

Notes:
- This scaffold contains layouts for Public, Patient, Doctor, and Admin.
- Pages are placeholders; wire up API and auth as next steps.
